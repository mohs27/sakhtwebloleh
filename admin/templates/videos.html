Instructions 

The admin panel is quite intuitive, so you will learn it easily.

1) Automated Contents

Videos are automaticaly added, if you have active sources/plugins.
A source is an external site like pornotube.com which the script will grab videos from.
A plugin is a sponsor source, which provides the videos to be retrieved.
(More sources/plugins are added once in a while. Read the Index News to be informed.)

You have to configure a cron job for automated contents to work.
Check the installation guide to set the cron up.

Videos can also be checked automatically, by setting another cron job. (automagick_cron.php?check=1)
Some videos, by technical reasons cannot be checked, so they will be automatically skipped by the cron job.
You can manually check a video by clicking on the "Check Video" Option or manually run the check cronjob for Tube 8, by clicking on "Tube 8 Check" .
Videos are updated or deleted depending on the result of the verification.

"Get Source's Videos" will manually run automagick_cron.php, to grab as many videos as it's configured.

2) Add New Videos

To add your own videos, click on "Add New Video" option. Pay attention to the tooltips, that explain the format of some fields.

You can add your own player if you want to. Just embedd the correspondient code pointing to your own player url.
You could also hotlink FLVs in your player if other sites allow it to.

3) Edit a Video

To edit a video, click on the "Edit Video" link. You will be able to change all values of the video.

4) Delete a Video

To delete a video, click on "Delete Video".
"Mass Delete" option will allow you to delete all videos of a specific source.

5) Filters and Options

Use them to browse, search and order videos as you wish. It will make your job easier.

6) Bug Reporting

I try to keep the script is constantly updated. Sometimes, little bugs appear and I need to fix them.
If you find any of them, please send me a report and I'll work on th as soon as I can. Bug Report

Important Information: Click to expand


If you are adding a video in FLV format, try to check if the video has the needed keyframes for Pseudo-Streaming to work fine. Other way, users won't be able to seek to other parts of the video.

If you are adding a video in MP4 format, try to check if the metadata is already located at the beginning of the file, so it can be played instantly.

If you don't know the situation of your case, test the video and, if you are not satisfied with the results (due to one of these problems), add the "Fix for Pseudo-Streaming" task to the video, and the script will take care of it. You will probably need to use tools like flvtool2 and ffmpeg to do it.

Smart Recategorization  
Smart Recategorization will attempt to assign each video, of the chosen group/category or selected videos, to one or more categories using the smart tags previously defined for each category and the title of the videos.

If no matches are found, the videos will not be moved, and if any matches are found, the video will be assigned to the new ones, without deleting the old ones (only 'Uncategorized' category is removed, in this case).

If you want the videos of the selected group/category to be recategorized within the same group, make sure you have checked the correct option in Settings Module > Advanced Settings > Smart Recategorization.
